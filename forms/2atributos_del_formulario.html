Atributos de Formulario






action

Si se omite action la accion se establece en la pagina actual

target

el atributo target especifica donde mostrar la respuesta que se recibe despues de enviar el formulario por defecto esta en _self lo que significa que la respuesta se abrira en la ventana actual

method

el atributo method especifica el metodo HTTP que se utilizara al enviar los datos del formulario

los datos de formulario se pueden enviar como variables URL con method="get"br son parte de la URL
o como transacciones posteriores HTTP con method="post" no son parte de la URL

el dato predeterminado es get

autocomplete

permite a los navegadores completar automaticamente los valores de los campos de formulario basandose en entradas anteriores de usuario autocomplete="on, off"

tambien se puede aumentar a elementos individuales como input, textarea y select

novalidate

significa que cualquier validacion del lado del cliente como: required, pattern, min max etc sera ignorada

el atributo novalidate desactiva la validacion automatica del navegador en los formularios permitiendo a los desarrolladores la validacion de los datos de entrada de manera personalizada
